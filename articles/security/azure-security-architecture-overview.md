# <a name="security-architecture-overview"></a>Güvenlik mimarisine genel bakış
Azure çözüm dağıtımlarda güvenli hale geldiğinde güçlü bir mimari temel sahip başarılı anahtarları biridir. Bu bilgiyle sormak için doğru soruları ve daha fazlasını sağa sorularınızın yanıtlarını bulmak için donatılmış öğrenerek gereksinimlerinizi daha iyi anlamak kullanabilirsiniz. Sağa doğru soruların yanıtlarını alma dağıtımlarınızı güvenlik en iyi duruma getirme doğru uzun bir yol gider.

Bu bölümdeki makaleler Azure güvenlik güvenli çözümleri oluşturmanıza yardımcı olacak mimarisine görürsünüz. Azure güvenlik en iyi yöntemler ve yaklaşımlar popüler koleksiyonu da dahil edilir. Bu aşamada, aşağıdaki makaleler sitemizi ve Azure güvenlik ekibi Web günlüğü güncelleştirmeleri düzenli olarak ziyaret ettiğinizden emin olun – vardır:

* [Bulut hazırlığı için veri sınıflandırması](azure-security-data-classification.md)
* [Microsoft Azure üzerinde uygulama mimarisi](security-application-architecture-on-azure.md)
* [Azure güvenlik en iyi uygulamaları ve desenleri](security-best-practices-and-patterns.md)
