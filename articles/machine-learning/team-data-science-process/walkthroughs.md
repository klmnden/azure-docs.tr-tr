# <a name="walkthroughs-executing-the-team-data-science-process"></a>Takım veri bilimi işlemi yürütülürken izlenecek yollar

Bunlar **uçtan uca izlenecek yollar** belirli senaryoları için takım veri bilimi işlemindeki adımlar gösterilmektedir. Bulut, şirket içi araçları ve Hizmetleri bir iş akışı veya oluşturmak için ardışık düzen birleştirmek nasıl gösteren bir **akıllı uygulama**. İzlenecek yollar göre gruplandırılır **platform** kullandıkları.


## <a name="walkthrough-descriptions"></a>İzlenecek yol açıklamaları

Ne Bu izlenecek yol örnekleri kendi ilgili platformlarında sağlamak, kısa açıklamaları şunlardır:


- [Azure Machine Learning çalışma ekranı](walkthroughs-aml-with-tdsp.md) Bu izlenecek yol öğreticilerine ve örneklerine doğal dil işleme ve sınıflandırma yapmak için Azure Machine Learning çalışma ile yapılandırılmış takım veri bilimi projeleri oluşturulacağını gösterir.
- [PySpark ve Scala kullanarak Hdınsight Spark izlenecek](walkthroughs-spark.md) Bu talimatlar PySpark ve Scala bir Azure Spark kümesinde Tahmine dayalı analiz yapmak için kullanın.
- [Hive kullanarak Hdınsight Hadoop izlenecek](walkthroughs-hdinsight-hadoop.md) Bu talimatlar Hive Tahmine dayalı analiz yapmak için bir Hdınsight Hadoop kümesi ile kullanın.
- [U-SQL'yi kullanarak azure Data Lake izlenecek](walkthroughs-azure-data-lake.md) Tahmine dayalı analiz yapmak için bu talimatlar U-SQL Azure Data Lake ile kullanın.
- [SQL Server](walkthroughs-sql-server.md) Bu talimatlar Tahmine dayalı analiz yapmak için SQL Server, SQL Server R Services ve SQL Server Python Hizmetleri kullanın.
- [SQL veri ambarı](walkthroughs-sql-data-warehouse.md) Tahmine dayalı analiz yapmak için SQL veri ambarı Bu izlenecek yollar kullanın.
