## <a name="overview"></a>Genel Bakış
Anomali Bulucu API'si, zaman içinde veri izleyip otomatik olarak sağ istatistiksel modelini uygulayarak benzersiz verilerinizden uyum sağlayan machine learning ile anomali algılama sağlar. Bu, kuruluşların veri kalitesini sağlamak için güvenilir hizmetleri sağlamak, iş olayları tanımlamak ve olağan dışı verileri el ile tanımlamak zor sorunları giderme olmadan iş yaklaşımlarını iyileştirmek sağlar.

### <a name="version-information"></a>Sürüm bilgileri
*Sürüm*: 1.0.0

### <a name="uri-scheme"></a>URI şeması
*Konak*: labsportalppe.azure-api.net  
*BasePath*: /anomalyfinder/v1.0  
*Düzenleri*: HTTPS

### <a name="tags"></a>Etiketler

* anomalydetectıon: Zaman içinde veri izlemenizi ve benzersiz verileriniz otomatik olarak sağ istatistiksel modelini uygulayarak uyum sağlayan machine learning ile anomali algılama API'si sağlar.
