# <a name="pause-and-start-your-power-bi-embedded-capacity-in-the-azure-portal"></a>Duraklatma ve Power BI Embedded kapasitenizi Azure portalında Başlat

Bu makalede, duraklatma ve Microsoft Azure üzerinde bir Power BI Embedded kapasite başlatma konusunda size yol göstermektedir. Bu Power BI Embedded kapasite oluşturduğunuz varsayar. Yüklemediyseniz, bkz: [Azure portalında Power BI Embedded oluşturmak kapasite](create-capacity.md) başlamak için.

Azure aboneliğiniz yoksa başlamadan önce [ücretsiz bir hesap](https://azure.microsoft.com/free/) oluşturun.

## <a name="pause-your-capacity"></a>Kapasite duraklatma

Kapasite duraklatma faturalandırılır gelen engeller. Bir süre için kapasite kullanılacak gerekmiyorsa kapasitenizi duraklatma mükemmeldir. Kapasite duraklatmak için aşağıdaki adımları kullanın.

> [!NOTE]
> Bir kapasite duraklatma içerik Power BI içinde kullanılabilir engelleyebilir. Çalışma alanları kesintisini önlemek için duraklatmadan önce kapasiteden atamasını kaldırmak emin olun.

1. [Azure portal](https://portal.azure.com/) oturum açın.

2. Seçin **tüm hizmetleri** > **Power BI Embedded** , kapasiteleri görmek için.

    ![Azure portal içindeki tüm hizmetler](media/pause-start/azure-portal-more-services.png)

3. Duraklatmak istediğinizden kapasite seçin.

    ![Azure portalı içinde Power BI Embedded kapasite listesi](media/pause-start/azure-portal-capacity-list.png)

4. Seçin **duraklatma** içinde kapasitesi ayrıntıları.

    ![Kapasite duraklatma](media/pause-start/azure-portal-pause-capacity.png)

5. Seçin **Evet**, kapasite duraklatmak istediğiniz onaylar.

    ![Duraklatma onaylayın](media/pause-start/azure-portal-confirm-pause.png)

## <a name="start-your-capacity"></a>Kapasite Başlat

Kullanım kapasitenizi başlatarak sürdürün. Kapasite başlangıç faturalama sürdürür.

1. [Azure portal](https://portal.azure.com/) oturum açın.

2. Seçin **tüm hizmetleri** > **Power BI Embedded** , kapasiteleri görmek için.

    ![Azure portal içindeki tüm hizmetler](media/pause-start/azure-portal-more-services.png)

3. Başlatmak istediğiniz kapasite seçin.

    ![Azure portalı içinde Power BI Embedded kapasite listesi](media/pause-start/azure-portal-capacity-list.png)

4. Seçin **Başlat** içinde kapasitesi ayrıntıları.

    ![Kapasite Başlat](media/pause-start/azure-portal-start-capacity.png)

5. Seçin **Evet**, kapasite başlatmak istiyor onaylar.

    ![Başlatmayı onayla](media/pause-start/azure-portal-confirm-start.png)

Tüm içerik için bu kapasite atanmışsa başlatıldığında kullanılabilir.
