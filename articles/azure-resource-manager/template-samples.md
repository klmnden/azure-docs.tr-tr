# <a name="azure-resource-manager-templates-for-management-features"></a>Yönetim özellikleri için Azure Resource Manager şablonları

Aşağıdaki tablo, Resource Manager tarafından sağlanan özellikler için Azure Resource Manager şablonlarının bağlantılarını içerir.

| | |
|-|-|
|**Rol atamaları**||
| [Kaynak grubu için rol atama](https://github.com/Azure/azure-quickstart-templates/tree/master/101-rbac-builtinrole-resourcegroup)| Yerleşik bir rol, mevcut bir kaynak grubu için bir kullanıcıya atar. |
| [Var olan sanal makine rolü atama](https://github.com/Azure/azure-quickstart-templates/tree/master/101-rbac-builtinrole-virtualmachine)| Yerleşik bir rol için bir kullanıcı için varolan bir VM'yi atar. |
| [Bazı sanal makineler için rol atama](https://github.com/Azure/azure-quickstart-templates/tree/master/201-rbac-builtinrole-multipleVMs)| Kullanıcılar birden fazla sanal makine için yerleşik roller atar. |
| [Azure aboneliği için rol atama](https://github.com/Azure/azure-quickstart-templates/tree/master/subscription-level-deployments/subscription-role-assigment)| Bir Azure aboneliği için bir kullanıcı bir rolü atar. |
|**Rol tanımı**||
| [Özel rol tanımını oluşturma](https://github.com/Azure/azure-quickstart-templates/tree/master/subscription-level-deployments/create-role-def)| Bir Azure aboneliğinde yeni bir rol tanımı oluşturur. |
|**Kaynak kilidi**||
| [Kilit kaynak grubu](https://github.com/Azure/azure-quickstart-templates/tree/master/subscription-level-deployments/create-rg-lock-role-assignment)| Bir kaynak grubu oluşturur ve geçerli bir **DoNotDelete** kaynak grubuna kilit. Bir kullanıcıya katkıda bulunan rolünü atar. |
